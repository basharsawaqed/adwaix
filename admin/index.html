🔐 تغيير كلمة المرور

كلمة المرور تُخزَّن كـ SHA-256 hash في المتصفح فقط.
يمكنك تغييرها في أي وقت.

📊 Cloudflare Web Analytics غير مُفعَّل

يعرض زوار الموقع، بلدانهم، مصادر الزيارات (بحث، مباشر، إحالة) — مجاني تماماً ويحترم الخصوصية.

  1. افتح Cloudflare Dashboard
  2. Analytics & Logs → Web Analytics → Add a site
  3. انسخ الـ token (32 حرف)
  4. افتح js/main.js وضع التوكن في متغير CF_ANALYTICS_TOKEN
  5. ادفع للـ Git — يبدأ التتبع فوراً
